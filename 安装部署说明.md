# 安卓端app部署文档

- 1.下载源码

  `git clone https://github.com/ssad2019/EE_easyeat_app`
  
- 2.使用Android Studio打开，导出生成apk
  - `Build -> Build APK(s)`
  - 生成的apk在`\app\build\outputs\apk\debug`中
